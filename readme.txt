这个Todo List的实战我们就做一个类似Tower任务管理的简版功能，不做任务指派和拖拽排序这些，就现实以下功能：

1,可以输入一句话创建一条任务---(已完成)
2,每条任务创建时可以选择一个颜色标识任务的重要层度---(已完成.bug:无默认option选项)
3,每条任务创建时可以指定一个任务的结束日期
4,可以输入一句话创建任务分组
5,每条任务创建时可以指定任务属于哪个分组
6,有一个列表可以展示所有已创建的任务，并且刷新以后还是可以显示。
7,本周在实现的时候任务列表可以先不用实现像Tower这样按任务组显示任务，直接用一个列表把任务按创建时间显示就可以了，只是每任务行上要显示任务的分组名称、结束日期、任务内容。
